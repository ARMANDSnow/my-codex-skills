# 岗位同义词与相近度

脚本实现位于 `scripts/standardize_job_title.py`。

|---|---|
| 电话销售 / 电销 / telesales / 电话销售专员 / 外呼销售 | 电话销售 |
| 销售代表 / 销售专员 / 业务员 / 销售顾问 / 销售经理 | 销售 |
| 面销 / 门店销售 / 导购 | 面销 |

| BD / 商务拓展 / 商务经理 | 商务拓展 |

| 行政专员 / 办公室助理 / 行政助理 / 文员 | 行政 |

## 相近度原则

- 销售子类之间：高相关，通常 0.8-0.9。
- 客服到销售：中相关，通常 0.5-0.55，可作弱判据加分，但单凭它一般不足以到强推荐。
- 市场/商务/渠道到销售：中到高相关，结合职责和指标判断。
- 行政到销售：低相关，通常 0.25。

未覆盖岗位保留原值，并使用相似度 0 处理。
