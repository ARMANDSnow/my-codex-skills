# 异常校验与稳定性规则

## 异常等级

- P0：拦截类，必须人工复核，不得自动强推荐。
- P1：降权类，降低推荐等级或进入待审核。
- P2：提示类，增加追问项，不直接淘汰。

## 一期异常

| 异常 | 条件 | 等级 |
|---|---|---|
| 年龄-工龄异常 | 年龄 ≤ 23 且正式工龄 > 4 年 | P0 |
| 年龄-工龄高异常 | 年龄 ≤ 25 且正式工龄 > 6 年 | P0 |
| 工龄开始时间异常 | 正式工作开始时小于 16 岁 | P0 |
| 时间倒置 | 开始日期晚于结束日期 | P0 |
| 同期多份全职 | 同期 ≥2 段正式工作 | P0 |
| 同期多段经历 | 同期 ≥3 段任何类型经历 | P0 |
| 全职实习重叠 | 正式工作与实习重叠 | P1 |
| 经历可信度低 | 可信度分 < 0.4 | P1 |
| 岗位跳跃异常 | 最近三段岗位相近度多数 < 0.3 | P1 |
| 近期频繁跳槽 | 最近 2 年 3 次变动或 3 段短任职 | P1 |
| 学历层级缺失 | 教育背景缺少学历 | P1 |
| 时间缺失/项目重叠/学校缺失 | 信息不足但可补充 | P2 |

## 稳定性分

履历稳定分 = 100 * (
0.35 * MIN(平均在岗月数, 42) / 42
+ 0.15 * MIN(中位在岗月数, 36) / 36
+ 0.20 * (1 - 短任职占比12)
+ 0.10 * (1 - 严重短任职占比6)
+ 0.20 * MIN(最近2段平均在岗月数 / MAX(更早经历平均在岗月数, 1), 1)
)

标签：

- 优秀：≥80
- 良好：60-79
- 一般：40-59
- 较差：<40

## Gap 分

Gap 分 = 100 * (
0.50 * (1 - MIN(近5年总Gap月数, 12) / 12)
+ 0.30 * (1 - MIN(最长Gap月数, 6) / 6)
+ 0.20 * Gap解释分
)

Gap 解释分：

- 有备考、进修、创业、项目、照顾家庭、生育、病休等明确说明：0.85
- 无说明：0.35
- 无 Gap：1.0

标签：

- 优秀：≥80
- 良好：70-79
- 一般：60-69
- 较差：<60
